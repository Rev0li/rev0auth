# Docs Index

Point d'entree unique pour eviter la dispersion.

## Learning fortress

This project is intentionally treated as a Rust learning fortress.
The codebase is used to onboard people step by step, with comments, branches, and commits that explain the why and the what.
The long-term goal is clarity first, then expansion.

## Where to start editing

- Learning story and commit path: `docs/learning/BookOfDev.md`
- Learning workflow and current branch state: `docs/learning/README.md`
- Execution roadmap and ticket order: `docs/roadmap-detailed.md`
- Consolidation notes and refactor targets: `docs/Next-Work.md`
- Ticket status and definitions of done: `docs/tickets-auth.md`

## Demarrage rapide

1. Lire `docs/roadmap-detailed.md` (ordre d'execution des etapes)
